# 🎯 Systems with OOP Dart

A collection of real-world systems built with **Dart**, applying core **Object-Oriented Programming (OOP)** principles such as encapsulation, inheritance, polymorphism, and abstraction.

---

## 📁 Systems Overview

| System | Description |
|--------|-------------|
| 🏦 **Bank System (ATM)** | Simulates ATM operations including deposits, withdrawals, and balance inquiries |
| 🚗 **Car Rental System** | Manages car availability, customer bookings, and rental pricing |
| 🛒 **E-Commerce System** | Handles products, shopping cart, orders, and customer management |
| 🏨 **Hotel Booking System** | Covers room booking, availability checking, and reservation management |
| 🍽️ **Restaurant Ordering System** | Manages menu items, customer orders, and billing |
| 📊 **Sales Management System** | Tracks products, sales transactions, and reporting |
| 📣 **Marketing Company System** | Models clients, campaigns, and marketing services |

---

## 🧠 OOP Concepts Applied

- **Classes & Objects** — Every system is modeled using well-defined classes
- **Encapsulation** — Data is protected using private fields and getters/setters
- **Inheritance** — Shared behavior is reused through base classes
- **Polymorphism** — Methods behave differently based on the object type
- **Abstraction** — Complex logic is hidden behind clean interfaces

---

## 🛠️ Tech Stack

- **Language:** [Dart](https://dart.dev/)
- **Paradigm:** Object-Oriented Programming (OOP)
- **IDE:** Any Dart-compatible editor (VS Code, IntelliJ, Android Studio)

## 🚀 Getting Started

### Prerequisites
Make sure you have [Dart SDK](https://dart.dev/get-dart) installed.

```bash
dart --version
```

### Run any system

1. Clone the repository:
```bash
git clone https://github.com/romisaa5/Systems_with_OOP_Dart.git
```

2. Navigate to a system folder:
```bash
cd Systems_with_OOP_Dart/"Bank System (ATM)"
```

3. Run the main file:
```bash
dart run main.dart
```
